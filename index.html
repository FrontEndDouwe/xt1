Tijdlijn ontwikkeling in interbrowsers

1990

De Eerste Browser

In het begin was het web statisch. Een webpagina werd gedownload en er veranderde niks aan. In 1990 maakte Tim Berners-Lee de eerste webbrowser. Webpaginas bestonden in deze tijd uit tekst met een klein beetje opmaak en links naar andere webpaginas. Het was een web aan informatie.

1994

Netscape en Javascript

In 1994 werd de browser Netscape gepubliceerd. Een van de voordelen van netscape was dat het gratis was en het nieuwe technische mogelijkheden had. Zo werd in netscape bijvoorbeeld het gebruik van cookies en Javascript geintroduceerd. .

1997

HTML

De eerste versie van HTML werd als draft gepubliceerd in 1993, echter werd in 1997 HTML3.2 gepubliceerd en werd dit door de 3WC als aanbeveling gegeven voor het maken van webpaginas. Later in 1997 werd HTML 4.0 uitgebracht.

2000

Abode Flash Player

Rond 2000 werd Flash geintroduceerd voor windows. Flash kon gebruikt worden om media of animaties af te spelen in de webbrowser. Dit was vooral populair voor het maken van bijvoorbeeld videogames en GUI's. Tegenwoordig word flash niet meer ondersteund.

2007

PHP 5

PHP is een scripttaal die voor het eerst uitkwam in 1994, echter werd het later pas populair. In 2004 kwam PHP5 uit echter, werd het in 2007 pas vaak gebruikt omdat het daarvoor instabiel was. PHP is vooral geschikt voor object georienteerd programmeren en het maken van webservers.

2009

Node.js

In 2009 Werd Node.js ontwikkeld. Dit is een open scource project waarmee je Javascript code kan gebruiken buiten de webbrowser dit zorgt ervoor dat je Javascipt ook buiten de client kan gebruiken en dat het niet alleen in de front end zit.

2011

Bootstrap

Bootstrap is een open source css framework dat is gepubliceerd in 2011, met als doel het makkelijker maken van webdevelopment. Bootstrap is een van de populairste open source projecten, zo heeft het meer dan 142.000 stars op github.

2014

HTML5

Sinds 2014 is HTML5 een W3C remommendation. In HTML5 zijn tenopzichten van de vorig versie meer tags toegevoegd. Dit zorgt ervoor dat er meer tags zijn voor de markup zoals de header, nav en footer tag. Ook zijn er tags toegevoegd voor het meer grafische content zoals de video, audio en canvas tags.
